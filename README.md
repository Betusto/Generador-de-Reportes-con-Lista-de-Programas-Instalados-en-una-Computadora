# Generador de Reportes con Lista de Programas Instalados en una Computadora
Consta de dos programas, uno encargado de extraer datos de los programas instalados en la PC que se esté ejecutando, esa información es mandada
a una base de datos, el otro programa se encarga de visualizar todos los documentos que estén guardados en la base de datos. 2019.<br/>
Lenguaje usado: Java.
